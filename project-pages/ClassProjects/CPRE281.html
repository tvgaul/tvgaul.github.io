CPRE 281 FPGA Dinosaur Game Back
I took CPRE 281(Introduction to Digital Logic) Fall of 2021 and learned the basics of digital logic. For my final project I choose to implement the dinosaur game from google when there is no internet on Logic and run it on an FPGA with 8 7 segment displays. 4 of these diplays displayed the dinosaur (the fair left vertical lines) and the obstacles to go jump or go under. These obstacles would go across the screen in a shift register and the jumping of the dinosaur was done with a simple state machine. The terrain generation was done by a larger state machine and "randomized" with the input of the dinosaur jumping. The score was displayed in Hex on two of the remaining seven segment displays and the high score was saved in the other one and saved in a register to be updated if it was beat. This project gave me some amazing experience with ditigal logic and FPGAs. The course also taught me how to use Quartus for FPGA development and testing.
